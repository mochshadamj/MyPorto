## 🚀 Penggunaan

1. **Navigasi**:
   - Gunakan tombol panah kiri/kanan
   - Klik indikator dot di kanan atas
   - Gunakan keyboard (← →)

2. **Mode Presentasi**:
   - Double klik untuk fullscreen
   - Tekan ESC untuk keluar fullscreen

3. **Form Kontak**:
   - Isi nama, email, dan pesan
   - Klik "Kirim Pesan" untuk menampilkan notifikasi
